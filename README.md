# Marketplace Backend APIs Project

# Description : Built a secure and scalable internal marketplace backend using Node.js, Express, and MongoDB with JWT-based authentication, robust validation, and production-ready architecture. The system supports item listing, searching, transactions between users, and role-based access control, along with max. test coverage using Jest and Supertest.

A secure and scalable internal marketplace backend built with:

- Node.js + Express
- TypeScript
- MongoDB
- JWT Authentication
- Jest Testing

## Features
- User Authentication
- Dashboard (Search, Filter, Pagination)
- Item Creation, Updation, Deletion API
- Transation Creation, Updation & Listing API
- Unit Testing

## Run Locally
npm install
npm run dev

## Run Tests
npm test
